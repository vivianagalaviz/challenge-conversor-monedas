## 📖 Descripción del Proyecto

Aplicación de consola desarrollada en **Java** que permite realizar conversiones de monedas en tiempo real utilizando la API de ExchangeRate.

El usuario puede seleccionar diferentes tipos de conversión desde un menú interactivo y obtener el valor actualizado según la tasa actual.

Este proyecto integra:

- Consumo de API REST
- Manejo de HttpClient, HttpRequest y HttpResponse
- Procesamiento de JSON con Gson
- Manejo de excepciones
- Uso de variables de entorno
- Programación orientada a objetos

# 🔐 Seguridad
La API Key no está expuesta en el código.  
Se utiliza una variable de entorno

# 🚀 Cómo ejecutar el proyecto

1. Clonar repositorio

2. Configurar variable de entorno:
EXCHANGE_API_KEY=tu_clave_aqui

3. Ejecutar clase Principal
